# Welcome! Let's learn to code together 🌟

Hi! You are about to build things with code for the very first time. That feels exciting, a little scary, and completely normal. This repo is your friendly classroom.

A **repo** (short for "repository") is just a folder that holds all the files for a project and keeps a history of every change. Think of it like a school binder with a memory: it remembers every page you ever wrote.

You don't need to know anything yet. You just need to show up and stay curious. Claude will be your patient tutor, one small step at a time.

## The only thing to do right now

Don't try to read everything. Do this one thing:

1. Open this repo in **Claude Code**. That is the tool where you chat with Claude right inside your project, like texting a helper who can see your files.
2. Copy the message below, paste it to Claude, and press Enter.

```
Hi Claude. I'm new to coding. Please read CLAUDE.md and be my tutor. Let's start with lesson 00.
```

That's it. Claude takes it from there and walks you through your very first lesson. 🚀

## Three golden rules of learning

1. **Mistakes are part of the game.** Every coder, even the best ones, breaks things all day long. A mistake is not failure. It is just the next clue.
2. **Type the code yourself.** Don't copy and paste big blocks of code. Typing it by hand is how your brain actually learns. Slow is smooth, and smooth is fast.
3. **Always ask "why?"** If something doesn't make sense, ask Claude to explain it again in a simpler way. There are no dumb questions here, ever.
