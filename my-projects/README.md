# Your Projects Live Here 🚀

Welcome to your own corner of this repo. This folder is **yours**.

Think of it like a sketchbook. A sketchbook is a book where you draw whatever
you want, and nobody expects your drawings to be perfect. This folder works the
same way, but for code.

## What this folder is for

- Building things you dream up.
- Trying stuff just to see what happens.
- **Breaking things on purpose.** Breaking things is how you learn what they do.
  You cannot damage anything important here, so go wild.

Every project you make lives in this folder. When you start something new, make
a fresh folder inside this one and give it a clear name (like `my-first-page` or
`snake-game`). That keeps each project tidy and easy to find later.

## Not sure what to build first?

That is totally normal, and it happens to everyone. Start with your very first
web page. It walks you through every step, one small piece at a time, so you are
never lost.

👉 Go to [`lessons/02-your-first-web-page.md`](../lessons/02-your-first-web-page.md)

You do not need to know anything yet. That is exactly what the lessons are for.
Have fun, and remember: mistakes are part of the deal. 💪
